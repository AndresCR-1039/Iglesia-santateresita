# Iglesia Santa Teresita — Sitio Web

Aplicación web para la gestión de contenido de una iglesia: noticias, sermones, horarios y panel de administración.

## Funcionalidades

- Página pública con información de la iglesia, horarios y noticias
- Sección de sermones
- Panel de administración con autenticación
- Gestión de noticias y sermones desde el panel admin
- Carga de archivos/imágenes

## Tecnologías

- Next.js (React)
- TypeScript
- Supabase (base de datos y autenticación)
- Tailwind CSS

## Estado del proyecto

Proyecto en desarrollo activo (versión inicial).

## Aprendizajes

Practiqué el desarrollo de una aplicación full-stack con Next.js, autenticación de usuarios, manejo de base de datos con Supabase, y protección de rutas administrativas.
